Guide page

Initial setup

When this add-on is installed (including reinstallation), the translated language depends on browser's locale. Also translation result is opened in a new tab.
There are items to change these settings are on the options page. (Red frame part in the next image)

Tips

Q. The translated language is different from the specified language.
A. It seems that the specification of Google translation is changed and it becomes the language used before. It may be resolved if you discard the cache and load the page on the Google Translate page. (Ctrl + F5 key)

Q. I want to erase the icon added to the address bar.
A. You can delete only the icon by right-clicking the icon.
