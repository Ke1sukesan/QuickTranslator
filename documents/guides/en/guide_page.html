Guides

Initial setup

When this add-on is installed (including reinstallation), the language after translation has not been specified. If you use the function in this state, the text or website will be translated into English and a notice will be displayed on the lower right.

If you set up "Behavior when translating" and "Translated language" on the option page of this add-on, the notification will not be displayed. (A red frame part of next image.)

Tips

Q. The translated language is different from the specified language.
A. It seems that the specification of Google translation is changed and it becomes the language used before. It may be resolved if you discard the cache and load the page on the Google Translate page. (Ctrl + F5 key)

Q. I want to erase the icon added to the address bar.
A. You can delete only the icon by right-clicking the icon.
